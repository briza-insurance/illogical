  • referenceSerialization(operand: string): string

    A function used to transform the stripped form of a reference into an annotated version of the reference. This function should be the inverse function of referenceTranform.

    referenceSerialization(referenceTransform("$Reference")) === '$Reference'

    E.g. "Reference" => "$Reference"
